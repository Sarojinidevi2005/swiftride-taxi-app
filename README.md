# 🚖 SwiftRide – Call Taxi Booking

SwiftRide is a responsive Taxi Booking website developed as part of the ApexPlanet Web Development Internship using HTML, CSS, and JavaScript.

## 📌 Internship Progress

### ✅ Task-1
- Basic HTML webpage
- CSS styling
- JavaScript interactivity
- Responsive navigation
- Taxi booking UI

### ✅ Task-2
- Responsive layout using Flexbox & CSS Grid
- Contact Form
- JavaScript Form Validation
- Dynamic Ride Planner
- Add/Delete/Complete ride tasks
- Task statistics (Total, Pending, Completed)
- Improved UI with animations and modern design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ✨ Features

- Responsive Navigation

- Ride Planner
